Reverse Engineering Task Notes
==============================

Script: analyze.py
Purpose: Processes a list of records and applies category-based multipliers.

Logic Summary:
- 'premium' category: value * 1.1
- 'standard' category: value * 0.9
- Adjusted values are rounded to 2 decimal places
- Total and average are computed from adjusted values

Expected Output Verification:
- Alice: 100 * 1.1 = 110.0
- Bob: 200 * 0.9 = 180.0
- Carol: 150 * 1.1 = 165.0
- Dave: 80 * 0.9 = 72.0
- Eve: 300 * 1.1 = 330.0
- Total: 857.0
- Average: 857.0 / 5 = 171.4
