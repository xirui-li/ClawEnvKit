  A production SQLite database used by a web application has become corrupted
  after an unexpected system crash. The main database file exists but appears
  to be in an inconsistent state. A WAL (Write-Ahead Log) journal file is
  present alongside the database, containing committed transactions that were
  not yet checkpointed into the main database file.

  Your task is to recover the database, merge the WAL journal, verify data
  integrity, and restore the application to a working state.
